DNS记录

  1. A记录，正向解析，域名toIP
  2. PTR记录，反向解析，IPto域名
  3. CNME记录，别名
  4. MX记录，mail.xxx.com，指向一个邮件服务器
  5. NS服务器，dns.xxx.com，指向域名有哪个dns服务器进行查询
  6. TXT记录,主机域名的查询

DNS查询方式

  1. 递归查询
  2. 迭代查询<-<-<-

  2. nslookup

  3. dig

  4. DNS配置文件地址：/etc/resolv.conf

  5. whois
